主题

“世界的48种可能”——SNH48 GROUP第八届偶像年度人气总决选

候选成员

SNH48（上海）、GNZ48（广州）、BEJ48（北京）、CKG48（重庆）所有符合参选条件的正式成员

入选席位

星光组：第1～16名
高飞组：第17～32名
梦想组：第33～48名

投票规则

购买第八届总决选投票EP《怦然心动》获得专属投票券即可通过以下方式投票：

官网投票

  1. 登录SNH48 GROUP第八届总决选专题；
  2. 登录账号；
  3. 激活投票券；
  4. 选择所支持的成员；
  5. 确认投票。

口袋48投票

  1. 打开口袋48 APP；
  2. 打开活动页面扫描二维码或输入序列号激活投票券；
  3. 选择所支持的成员；
  4. 确认投票。

官方微信公众号投票

  1. 关注官方微信公众号；
  2. 选择菜单“第八届总决选”；
  3. 扫描投票二维码或输入投票序列号，激活并获得投票权；
  4. 选择所支持的成员；
  5. 确认投票。

* 每张投票券对应一个投票序列号，激活投票序列号后可获得相应投票权（标准版一个序列号可激活10个投票权，应援版一个序列号可激活480个投票权）

特别说明：

  1. SNH48偶像年度人气总决选活动，不接受任何商业目的的投票,也不接受以商家名义进行的投票。
  2. 上海丝芭文化传媒集团及各下属机构员工，工作人员及其直系亲属均不得参与投票。
  3. 本活动解释权属上海丝芭文化传媒集团。
  4. 违规投票者将被取消所投票数。
  5. 同一帐户不得同时从不同IP地址登陆，否则所投票数将被视为无效。
  6. 秉持鼓励新人的原则，本届总决选将增设"新人激励机制"：首次参与总决选的新人成员根据排名从高至低前7名新人将颁发潜力新人奖项、获得新人专属生日公演等激励。本机制旨在让新成员在鼓励中不断成长与进步。
  7. 本届年度总决选演唱会还将新增"现场淘汰制"：与上一年仅148人前往演唱会活动现场不同，本年度总决选第一阶段的演唱会由SNH48 GROUP全员参与共同表演。在第二阶段的总决选排名现场公布环节开启前，现场将设置148个席位，主持人公布此次总决选最终位列TOP148的成员名单，名单内的成员依次从后台等候区进入星梦通道，登台上舞台亮相入座并等待最终排名的发表；而未入列TOP148名的成员，则将在所在队伍成员全部上场之后，统一前往告别通道，乘坐巴士离开演唱会现场。
