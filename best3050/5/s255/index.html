SNH48 GROUP年度金曲大赏 BEST50 REQUEST TIME歌曲总决选演唱会首度移师广州举办。来自SNH48及姐妹团BEJ48、GNZ48、SHY48、CKG48五团成员将共同参与。现场将揭晓由粉丝们票选出的50首人气歌曲与演绎阵容，公布全新7人小组合阵容，并发布SNH48 GROUP自组建以来最为重大的官方发表。
演出信息：SNH48第五届年度金曲大赏 BEST 50 REQUEST TIME歌曲总决选演唱会
时间：2019年1月19日 17:30
地点：广州体育馆（广州白云区白云大道南783号）
1月19日晚，由独家战略音乐合作平台咪咕音乐全程支持，独家互联网直播平台咪咕音乐、腾讯视频及海外独家直播平台YouTube联合呈现的SNH48 GROUP 第五届年度金曲大赏REQUEST TIME BEST 50 演唱会在广州体育馆圆满落幕。主持人在现场一一揭晓了由粉丝们投票选出的50首金曲及阵容，以及发布了SNH48全新7人小组名单。
 
延续速报与中报的势头，由SNH48戴萌和莫寒搭档演绎的歌曲《就差一点点》最终摘得了本届年度金曲大赏第一名。由BEJ48 Team B共同演绎的16人曲《笨》夺得了16人曲的第一名。这两首年度金曲及其演出阵容将获得定制MV的专属奖励，《就差一点点》Center成员戴萌和《笨》Center成员段艺璇作为代表获得荣誉奖杯。
偶像产业进入全新发展阶段
邀您共同见证战略大重组
随着以彰显互联网时代全民参与感的综艺节目造星模式的出现， 中国偶像产业的发展进入了中盘阶段，出现了才艺水准提升，迭代速度加快的新趋势， 为了顺应市场发展并在新的发展环境下，和粉丝朋友们共同努力，使SNH48 GROUP继续成为中国女团市场具有突出影响力的领导品牌，上海丝芭文化传媒集团决定对现有的团队设置，模式设定和资源分配进行重大调整，实施全团战略大重组并将于第五届年度金曲大赏演唱会的现场正式公布，在此诚挚邀请各位关心SNH48 GROUP发展的粉丝朋友们参与，和偶像成员们一起现场见证战略大重组这一重大历史时刻。
此次战略大调整，将伴随一系列全新举措的发布，SNH48 GROUP将从目前的单一运营模式朝多模式并举方向转型， 加快和加大优质偶像的人才培养，大幅提高头部成员的推广力度，继续提升娱乐内容的生产能力，其核心是持续提升粉丝体验，实现优质偶像养成和造星事业的新飞跃。面对中国新生代偶像产业进入中盘发展阶段的新局面，丝芭文化将进一步明确所肩负的发展责任，全力打造被社会大众认可的优质偶像和优秀艺人。
首度移师广州，唱享金曲激情
第五届SNH48 GROUP年度金曲大赏演唱会首度移师广州举办，50首金曲即将响彻羊城，将青春正能量与美丽的广州火热交织，向更多人传递属于我们的音乐与梦想，为粉丝们打造一场精彩绝伦的音乐饕餮盛宴。
玩法花样升级，金曲组别竞争全新推出
SNH48 GROUP年度金曲大赏BEST50 REQUEST TIME再度惊喜来临，本届金曲大赏演唱会依旧秉承着“用户即制作人”的根本理念，除了本年度最受欢迎50首歌曲及其演出阵容均由粉丝通过投票决选之外，更有全新升级的“金曲组别”竞争加入，获票数最高的UNIT曲和16人曲将成为年度金曲，其余48首歌曲根据粉丝投票票数分为3个等级，从低到高依次顺序为“星辰组”、“皓月组”、“曜阳组”，每个组别的入选歌曲数量各有不同。本届金曲大赏演唱会投票内容更加丰富，演出阵容更加强大、舞台呈现更多惊喜，更多无法预知的感动将在现场上演......
舞美震撼升级，多重配置由您解锁
第五届年度金曲大赏演唱会舞台将会由多块舞台区域组合而成，并且根据不同等级的歌曲获得相应的舞台使用空间及舞美设计，根据粉丝投票解锁相应组别配置。本届金曲大赏演唱会将呈现耳目一新的舞台，致力让观众共享这场满足全感官的视听盛宴！
奖励重磅升级，专属荣耀由您打造
本届年度金曲大赏演唱会最终获票数最高的16人曲及UNIT曲将成为年度金曲并定制专属MV，同时，年度金曲将连同进入“曜阳组”的七首金曲收录于SNH48 GROUP春季EP B50特别纪念版。本届金曲大赏获得年度金曲及进入“曜阳组”金曲的Center成员将作为代表获得荣誉奖杯，一起见证荣耀的瞬间！！快来为您喜欢的歌曲和阵容投上神圣的一票！
SNH48 GROUP全新7人小组合，阵容由您决定！
作为本届金曲大赏演唱会中备受关注的“亮点”之一，为了打造颜值与实力相结合，引领偶像新风潮的优秀偶像组合，给观众们带来更新鲜的组合阵容搭配及更精彩的舞台表现，SNH48 GROUP将推出全新7人小组合。通过对偶像们的舞台表现能力、把控能力及感染力等多方面综合考虑后，在一众优秀的偶像中选拔出全新小组合候选阵容，由粉丝通过投票选出票数最高的7名成员以小组合形式出道。观众们可以通过SNH48 GROUP第五届年度金曲大赏演唱会专题页、口袋48、SNH48微信公众号，为您支持的偶像投票！
 
SNH48 GROUP年度金曲大赏BEST50 REQUEST TIME歌曲总决选演唱会将于2019年1月19日17:30在广州体育馆隆重举行！凡是购买专属投票EP《魔女的诗篇》、《终无艳》、《天空信》的粉丝，除了在第五届金曲大赏结束次日（2019年1月20日）可使用该EP内包含的握手券与成员握手外，还可于12月中旬起参与SNH48 GROUP各地剧场举办的专属答谢握手会。
